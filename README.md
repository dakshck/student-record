# student-record
A simple CLI student management system written in C. yeah but obvi, the learning project.

---

### Features

- Add Students
- Calculate's Student grade with marks
- List students 
- Formatted Output
- Delete students
- NUll data checks

---

### Concepts that i got my hands on

- `typedef structs`
- Using structs
- `strcspn` to remove newline (was fun to learn)
- got comfortable with shifting 2d arrays
- array handling with pointers

---

### Future improvements

- more decisions under add, del, and list students
- more input fields per student
- aesthetic looking dashboard (on this one too yeah)

---

### How to Compile and Run

**With gcc:**
```bash
gcc student-manager.c -o rec && ./rec
```

**With Clang:**
```bash
clang student-manager.c -o rec && ./rec
```
